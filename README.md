# 🎟️ FE-Tixin

FE-Tixin adalah aplikasi frontend berbasis web untuk sistem pemesanan tiket yang dibangun menggunakan teknologi modern seperti Next.js dan React. Project ini berfokus pada performa, skalabilitas, serta pengalaman pengguna yang optimal.

---

## 🚀 Project Overview

FE-Tixin menyediakan antarmuka pengguna untuk sistem ticketing, memungkinkan user untuk melakukan pemesanan tiket secara efisien dengan UI yang responsif dan modern.

Project ini juga mengimplementasikan best practice dalam pengembangan frontend seperti state management, data fetching optimization, dan modular component structure.

---

## 🧰 Tech Stack

- **Next.js 15** – React Framework (App Router)
- **React 19** – UI Library
- **TypeScript** – Static typing
- **Tailwind CSS** – Utility-first CSS framework
- **Next UI / Hero UI** – UI components
- **TanStack React Query** – Data fetching & caching
- **Axios** – HTTP client
- **React Hook Form** – Form management
- **Yup** – Schema validation
- **NextAuth.js** – Authentication

---

## ✨ Features

- 🔐 User Authentication (Login & Register)
- 🎫 Ticket Booking System
- 📊 Dashboard Management
- ⚡ Optimized Data Fetching (React Query)
- ✅ Form Validation with Yup
- 🎨 Responsive & Modern UI
- 🔄 State Management Best Practices
